Frequently Asked Questions

Authentication & User Management

How do I log into the admin panel?
Use your registered username and password on the admin login page. If you encounter an invalid login message, verify your credentials.
What should I do if I forget my password?
Currently, the system doesn't have a password reset link. Contact an administrator to reset your password manually.
Can I register a new admin user myself?
No. New admin users are created via the signup page and verified through an OTP sent to your email.
What is an OTP and how is it used?
OTP (One-Time Password) is a randomly generated 6-digit code sent to your email during signup for identity verification.

Plans & Subscription Management

How do I create a new subscription plan?
Go to the Manage Plans page, fill in the required details like name, price, API call limits, validity, and submit.
Can I edit an existing plan?
Yes — click the edit option next to a plan in the Manage Plans page, modify the details, and save.
How do I deactivate or activate a plan?
Use the Deactivate or Activate button next to each plan to toggle its status.
What happens when a plan expires?
Expired plans are automatically marked inactive based on their expiry date but can be reactivated manually.

Coupon & Discount Management

How do I create a new coupon code?
Go to the Create Coupon page, enter a unique code, discount percentage, valid from and to dates, and submit.
Can I edit or delete an existing coupon?
Yes — edit options are available next to each coupon. If a coupon is no longer needed, delete it from the list.

API Usage & Credit Management

How do I monitor API usage?
Navigate to the API Usage page in your admin panel to view API call statistics, usage limits, and credit consumption.
Can credits be adjusted manually?
Yes — admins can manually update credit balances for any user under the Credit Management section.
